### Ninjadogg SVG Editor
## An experiment in AGI code implementation given human-designed architecture

# Description:
Ninjadogg SVG editor is a final project for my Object-Oriented Analysis and Design class. This application will
allow a user to to create simple Scalable Vector Graphic art on a Canvas similar to tools like draw.io.

# NOTE:
All architectural design choices were chosen by me, Sabir Saklayen. Gemini 2.5 Pro and Claude Opus 3.5 were used
to aid in code implementation with varied results. This project was completed in Summer of 2024.
